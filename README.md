# mayatrail-web

The official website for **MayaTrail** — cloud attack emulation for detection engineers.

**Live at [mayatrail.tech](https://mayatrail.tech)**

## About

This repository contains the source code for the MayaTrail website — a static site (HTML/CSS + a little React) served via nginx in Docker, deployed on AWS. The site code lives in [`site/`](site/).
